QuizMe: Personalized Learning Platform

QuizMe is a web-based application designed to enhance learning through active recall and personalized question generation using Generative AI. It empowers users to study smarter by creating tailored quizzes based on their input materials.

  • Tech Stack: HTML, CSS, JavaScript, React, Tailwind CSS
  • Features: AI-generated quizzes, user progress tracking, responsive design
  • Purpose: Facilitate effective self-study with adaptive learning techniques

CryptoQuest: Educational Cryptography Game

CryptoQuest is an interactive web-based game designed to teach high school students about cryptography through Caesar and Vigenère ciphers. Built with HTML, CSS, and JavaScript, it features a spy-themed narrative, a 5-minute timer per level, high-score tracking, and sound effects to engage learners in STEM education.

  • Tech Stack: HTML, CSS, JavaScript
  • Features: Two cipher levels, hints system, timer, high-score system, responsive design
  • Purpose: Promote STEM learning through gamified cryptography challenges

Research Contributions

Active Recall via GenAI-Generated Questioning: A Personalized Self-Study Approach with QuizMe

This study presents QuizMe, a platform leveraging Generative AI to generate active recall questions from students’ materials. An experiment with 67 HUST students showed significant score improvements (from 6.985 to 8.791, p < 0.001) and 85% rated usability as “good” or “excellent.” Students praised its convenience but noted mobile interface limitations. Authors: Nguyen Phuong Ha, Nguyen Khanh Thien, Nguyen Duc Hai Dang, Dao Thanh Manh, Pham Duy Bao.

  • Published: Hanoi University of Science and Technology, 2025
  • Key Findings: AI-driven active recall improves retention by 25.8% compared to traditional methods
  • Methodology: Controlled experiment with 67 students, paired t-tests, thematic analysis
